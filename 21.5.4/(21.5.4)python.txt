정형데이터
반정형데이터 : 텍스트
비정형데이터

HDD, SSD(보조기억장치): 비휘발성
RAM(memory, 주기억장치) : 휘발성, 입력장치 및 출력장치들과 연결
CPU(전산장치) : 임시기억장치(register)-32bit(2**32), 64bit(2**64), 
프로세서 : CPU, 초당 처리 속도
프로세스 : 실행 중인 프로그램, 운영체제가 관리하는 실행 단위, PIB(프로세스 아이디)를 부여하여 관리
캐시(Cache) : 데이터나 값을 미리 복사해 놓는 임시 장소

표준입력장치(stdin) : 키보드, 마우스, ...
표준출력장치(stdout) : 모니터, 스피커, ...

기본 입력은 str으로 인식하므로 형 변환 과정이 필요하다

\n : new line
\t : tab
\a : alert
\b : back

%d : decimal(10진수)
%s : string
$f : float

데이터와 데이터 구분 : tab, space

제어문

반복문 : for, while

조건문 : if, else, elif, continue, break

던더(__함수__) = 매직 매서드
멤버 함수 = 매서드

dir() 함수를 통해 사용할 수 있는 매직 매서드와 멤버 함수를 확인할 수 있다.
